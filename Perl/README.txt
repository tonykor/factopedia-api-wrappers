README FILE FOR PERL API

To run the Perl program, first install Perl Installer as per your platform. After installation use CPAN feature of Perl to install libraries used in this API, viz. 
1) LWP::UserAgent
2) LWP::ConsoleLogger::Easy
3) JSON
4) MIME::Base64

you can use following command to install via Perl's CPAN feature:

cpan install LWP::UserAgent
